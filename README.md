# Atividade do Mario - Portfólio Inicial (19/05) 🚀

Este é o meu primeiro projeto aplicando os conceitos de HTML5 estrutural, estilização ágil com **Bootstrap 5** e controle de versão utilizando o **Git/GitHub**. A página foi desenvolvida exclusivamente para atender aos critérios da "Missão Prática" passada pelo professor Mario.

*Nota de esclarecimento: O trabalho originalmente era para ser feito em equipe de 2 ou 3 pessoas, mas resolvi fazer o projeto totalmente sozinho.*

---

## 🎮 Tema do Site
O tema escolhido para a página foi um **Portfólio Pessoal** bem sincero, listando o que seriam os meus projetos atuais e futuros. E para deixar a seção inicial com estilo, usei uma imagem de peso: **Dota 2**, o melhor jogo.

---

## 📑 Requisitos Técnicos Atendidos
Abaixo estão listados todos os componentes do Bootstrap que foram consultados na documentação oficial, copiados, colados e devidamente adaptados para o código:

1. **Navbar (Menu de Navegação):** Cabeçalho responsivo no topo da tela com links que levam direto para as seções certas através de âncoras (`#inicio` e `#meus-projetos`).
2. **Container e Grid:** O conteúdo da página está envelopado de forma que nunca fique colado nas bordas da tela, dividindo-se de forma inteligente e responsiva em colunas (`col-md-6` e `col-md-4`).
3. **Cards:** Adicionada uma seção dedicada a exibir três "cards" simulando o meu portfólio de projetos (se eu tivesse algum pronto ainda).
4. **Botões:** Uso dos botões estilizados oficiais do framework, aplicando as cores `btn-primary` (azul) e `btn-dark` (preto).
5. **Espaçamento (Spacing):** Aplicação de utilitários de margem e preenchimento (como `py-5` e `mb-5`) para garantir que o layout respire e não fique tudo amontoado.
6. **Comentários no Código:** Embora tenha feito o projeto sozinho, adicionei comentários explícitos em todas as seções do arquivo HTML (``) assinando o meu nome, cumprindo a exigência de organização da atividade.
